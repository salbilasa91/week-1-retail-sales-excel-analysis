# Week 1 - Retail Sales Excel Analysis

## Project Overview
This project was completed as part of the LogicStack Data Analysis Internship (Week 1).

The objective of this project is to perform basic data cleaning, formatting, analysis, and visualization using Microsoft Excel on a retail sales dataset, then to understand how data is prepared for analysis.

## Tools Used
- Microsoft Excel

## Dataset
- retail_sales_dataset.csv

## Tasks Completed
- Dataset understanding
- Data cleaning and formatting
- Data quality checking
- Basic Excel formulas
- Sorting and filtering
- Basic Charts and visualizations 
- Observations

## Key Findings
- Clothing has the highest quantity sold.
- Electronics generated the highest total sales.
- The average quantity sold per transaction is 3 products.
- There are no blank cells in the dataset.
- All Total Amount values match the calculated values.
- Customers aged to 41 to 60 (Senior Adults) generated the highest total sales.
